# VocEdit 

VocEdit is [KurrawongAI](https://kurrawong.ai)'s dedicated vocabulary editing application.

It allows non-technical staff to edit the contents of [Semantic Web](https://en.wikipedia.org/wiki/Semantic_Web) vocabularies created according to the [VocPub Profile of SKOS](https://linked.data.gov.au/def/vocpub/spec) data model without having to type code.

We normally configure VocEdit to work with client's particular set of vocabularies and this allows us to add in management functions too, but out-of-the-box, VocEdit can edit any given VocPub-conformant vocabulary in a file and save the results back to a file.
