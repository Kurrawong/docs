# Prefix Lookup 

The Prefix Lookup tool online at <https://tools.kurrawong.ai/prefixes> allows you to search for an RDF namespace based 
by using a common prefix for it, for example `geo` is commonly used for the GeoSPARQL Ontology with a namespace of 
`http://www.opengis.net/ont/geosparql#`.

The tool indicates where it finds result matches.
